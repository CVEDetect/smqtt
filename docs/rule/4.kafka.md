# KAFKA 节点详解


## 设计参数

|  参数   | 说明  | 必传  |默认值  |
|  ----  | ----  |----  |----  |
| ruleType  | 规则节点类型 |是 |KAFKA  |
| script| 脚本 |否 |无  |


## yaml配置

   ```
        smqtt:
            rules: # 规则引擎相关配置
              - ruleName: 过滤
                chain: # 规则链
                  - ruleType: KAFKA  # KAFKA过滤器
                    script:  '{"topic":"${topic}","msg":"${msg.test}","qos":${qos}}'
   ```
