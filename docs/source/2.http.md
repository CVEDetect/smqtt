# HTTP 数据源


## 配置详解

|  参数   | 说明  | 必传  |默认值  |
|  ----  | ----  |----  |----  |
| source  | 类型 |是 |HTTP  |
| sourceName| 数据源名称 | 是|无  |
| sourceAttributes| 额外参数 | 否|无  |

```markdown
smqtt:
  sources: # 配置数据源sources
    - source: HTTP
      sourceName: http
      sourceAttributes:
        url: http://www.baidu.com
        port: 80
        headers:
          Authorization: token value
```

> 仅仅支持POST请求，Content-Type : application/json
