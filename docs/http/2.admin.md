# 管理页面设置

## main启动参数详解

    ```
         BootstrapConfig
              .HttpConfig
              .builder()
              .enable(true)
              .accessLog(true)
              .admin(BootstrapConfig.HttpAdmin.builder().enable(true).username("smqtt").password("smqtt").build())
              .build()
    ```
## yml配置

        ```
          smqtt:
            logLevel: DEBUG # 系统日志
            http: # http相关配置 端口固定60000
              enable: true # 开关
              accessLog: true # http访问日志
              ssl: # ssl配置
                enable: false
              admin: # 后台管理配置
                enable: true  # 开关
                username: smqtt # 访问用户名
                password: smqtt # 访问密码
        ```
